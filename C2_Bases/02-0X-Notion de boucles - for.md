# Notion de boucles - for

Que va afficher à l'exécution chacun des groupes d'instructions ci-dessous ?

```cpp
//1
for (int i = 0; i <= 5; i += 2) {
    printf("%d ", i);
}
```

<details>
<summary>Solution</summary>

0 2 4

</details>


```cpp
//2
for (int i = 1; i != 16; i *= 2) {
    printf("%d ", i);
}
```

<details>
<summary>Solution</summary>

1 2 4 8

</details>
